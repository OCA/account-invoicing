This module is a glue module between the
partner_invoicing_mode_at_shipping module and the
partner_invoicing_mode_fourteen_days one.

As invoices are generated in draft when the shipping is done and as the
next_invoice_date on the partner level should be taken into account for
fourteen_days flow.
