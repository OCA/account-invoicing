=================================================
Automatically select invoicing partner on invoice
=================================================

On an invoice, when selecting a partner of any other type than 'invoice',
replace the partner by an invoice contact if found.

Installation
============

There are no specific installation instructions for this module.
Just click on install buttom

Configuration
=============

There are no specific configuration instructions for this module.
After you need to configure on partner profile, one contact partner as "invoice address"

Usage
=====

This module need base account module.
When create invoice for the partner it will take the selected contact address.
