=========================================
Account Invoice Default Company Bank Sale
=========================================

This module depends from sale_stock and allows to set partner_bank_id as 
default_company_bank_id when the invoice will be created starting from a sale order or from a stock.picking 

Installation
============

This module depends on:

* sale_stock

Configuration
=============

There is nothing to configure.
