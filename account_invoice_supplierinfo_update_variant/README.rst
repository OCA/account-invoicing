=============================================================
Update Supplier Info of product variant from Supplier Invoice
=============================================================

This module allows in the supplier invoice, to automatically update all
products variants informations whose purchase informations on the line are different
from the supplier informations, defined on the product variant form.

It creates a new supplier information line if there is not or it updates the
first.
